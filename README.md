Welcome to my Machine Learning journey! 🌟 This repository documents my 100 Days of ML Challenge, where I am building a robust, industry-ready portfolio by getting my hands dirty with real-world datasets.

📦 Current Focus: The Brazilian E-Commerce Dataset
I kickstarted this challenge by diving into the Brazilian E-Commerce Public Dataset by Olist.

My approach for this phase:

Exploratory Data Analysis (EDA): Understanding the hidden patterns in the data.

Feature Engineering: Crafting meaningful features to get the data ML-ready.

Model Training: Wait, what about scaling and encoding? 🤔

Pro Tip: While some suggest doing encoding and scaling right away, I keep these steps strictly inside the training phase (often using pipelines) to prevent data leakage.
